# 🎬 YouTube Downloader Bot

Un bot Telegram qui télécharge des vidéos YouTube et les envoie directement dans le chat.

## Fonctionnalités
- Télécharge des vidéos YouTube
- Envoie la vidéo directement dans Telegram
- Limite de 10 minutes par vidéo

## Installation

1. Clone le projet
2. Installe les dépendances :
   npm install

3. Copie .env.example en .env :
   cp .env.example .env

4. Remplis le token Telegram dans .env

## Démarrage

   npm start

## Utilisation

1. Démarre le bot avec /start
2. Envoie un lien YouTube
3. Reçois la vidéo !
